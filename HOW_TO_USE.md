# AI میل پلانر: ہوسٹنگ اور استعمال کی مکمل گائیڈ

اپنی ایپ کو لائیو کرنے اور دوسروں کے ساتھ شیئر کرنے کے لیے ان آسان اقدامات پر عمل کریں:

## 1. GitHub پر لائیو کرنا (GitHub Pages)
یہ سب سے پہلا اور بہترین طریقہ ہے:
1.  **اکاؤنٹ بنائیں:** [github.com](https://github.com) پر جائیں اور لاگ ان کریں۔
2.  **نیا ریپوزٹری:** "New" پر کلک کریں، نام دیں (مثلاً `my-recipes`) اور "Create" کریں۔
3.  **فائلیں اپ لوڈ کریں:** "uploading an existing file" پر کلک کریں اور اپنے اس فولڈر کی تمام فائلیں وہاں ڈال کر "Commit" کر دیں۔
4.  **سیٹنگز:** ریپوزٹری کی **Settings > Pages** میں جائیں۔ "Branch" میں `main` سلیکٹ کر کے "Save" کریں۔
5.  **لنک:** 2 منٹ بعد آپ کو ایک لنک ملے گا، اسے آپ کہیں بھی چلا سکتے ہیں۔

## 2. Vercel پر لائیو کرنا (Vercel)
اگر آپ چاہتے ہیں کہ ایپ اور بھی تیز چلے:
1.  [vercel.com](https://vercel.com) پر جا کر GitHub کے ذریعے سائن ان کریں۔
2.  **Add New Project:** اپنی GitHub والی ریپوزٹری کو سلیکٹ کریں۔
3.  **Deploy:** بٹن دبائیں اور آپ کی ایپ ایک پروفیشنل ہوسٹڈ لنک پر لائیو ہو جائے گی۔

## 3. WhatsApp پر شیئر کرنا
1.  اپنا لائیو لنک (GitHub یا Vercel والا) کاپی کریں۔
2.  WhatsApp پر دوستوں کو میسج میں یہ لنک بھیجیں۔
3.  انہیں کہیں کہ اس لنک کو اوپن کر کے تھری ڈاٹ مینیو سے **"Add to Home Screen"** کر لیں تاکہ یہ ان کے موبائل میں ایپ کی طرح انسٹال ہو جائے۔

## 4. موبائل پر انسٹال (PWA) کیسے کریں؟
- **Android:** لنک اوپن کریں، مینیو (⋮) میں جائیں اور **"Install App"** پر کلک کریں۔
- **iPhone:** لنک اوپن کریں، 'Share' بٹن دبائیں اور **"Add to Home Screen"** منتخب کریں۔
